# Full-Stack Platform

Production-ready full-stack platform built with React, Node.js, Nginx, PostgreSQL, Redis, and Docker Compose.

## Architecture
Browser - Nginx - Node API - PostgreSQL / Redis

## Services
- Frontend: React (static build)
- Backend: Node.js HTTP API
- Reverse Proxy: Nginx
- Database: PostgreSQL
- Cache: Redis
